# Booking Template (PHP + HTML + CSS)

Proyecto listo para poner en **htdocs** (XAMPP/MAMP/WAMP) y abrir en VS Code.

## Estructura
- `index.php` (flujo de 3 pasos con session)
- `assets/css/style.css`
- `assets/img/hero.jpg` (imagen placeholder, cámbiala por la tuya)

## Cómo correrlo (XAMPP)
1. Copia la carpeta `booking_template_php` dentro de:
   - Windows: `C:\xampp\htdocs\`
   - macOS (MAMP): `Applications/MAMP/htdocs/` (depende de tu setup)
2. Enciende Apache.
3. Abre en el browser:
   - `http://localhost/booking_template_php/`

## Notas
- Esto es un template (demo). En la confirmación no guarda en base de datos.
- Si quieres, te lo conecto a MySQL (phpMyAdmin) y le pongo tabla de reservas.
